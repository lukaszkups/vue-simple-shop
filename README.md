# vue-simple-shop

## Project setup

Install dependencies:

```
npm install
```

Run API server (REQUIRED TO RUN SPA CORRECTLY):
```
npm run api-server
```

Run front-end SPA (in separate tab):
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
